### 技术栈

- Vue：^2.6.10
- Vue-cli：^3.10.0
- Vue-router：^3.0.3
- Element UI：^2.4.5
- Sass

### 本地运行

```
git clone https://github.com/ChenxiiCheng/Vue-Admin.git
cd Vue-Admin
npm install  // Or use yarn
npm run start
```
